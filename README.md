# RRT-python
Implementation of RRT algorithm with python

## Usage
Just run below command at terminal
```
git clone https://github.com/dohyeoklee/RRT-python
cd RRT-python/src
python rrt.py
```
Dependency: 

-python 3.8.5

-numpy: 1.19.2

-matplotlib: 3.3.2
